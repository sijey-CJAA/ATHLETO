Universal Fit Notes:

  • Running: Size down for compression, true size for looser fits
  • Basketball: Size up for jerseys, true size for shorts
  • Training: True size for weightlifting, +1 for HIIT/cardio
  • Yoga: True size for compression, +1 for relaxed fit
  • Teens: Consider growth room in sleeves/legs
  • Women: Hip measurement most crucial for bottoms
  • Men: Shoulder width key for tops

Athletic Shoe Sizing Notes:

Running Shoes:

  • ½ size up from casual shoes
  • Thumb's width at longest toe
  • Consider arch type

Basketball Shoes:

  • True to size for performance
  • Ankle support crucial
  • Wear game socks when fitting

Training Shoes:

  • Snug but not tight
  • Lateral support important
  • Consider orthotics if needed

SPORT-SPECIFIC SIZE ADJUSTMENTS

🏃‍♀️ Running Gear

Top Fit: Compression (true size) / Loose (+1 size)

Bottom Fit: 5-7" inseam shorts / full-length tights

Key Measurement: Chest for tops, waist for shorts

🏀 Basketball Gear

Jerseys: +1 size for mobility

Shorts: 9-11" inseam preferred

Key Measurement: Shoulder width for jerseys

🏋️‍♂️ Training Gear

Weightlifting: True size for support

HIIT/Cardio: +1 size for airflow

Key Measurement: Chest & hip flexibility

🧘‍♀️ Yoga Gear

Tops: True size for compression

Bottoms: Stretch fabric, no size up needed

Key Measurement: Full range of motion

DEMOGRAPHIC FIT GUIDE

♀ Women's Fit

Top Size Tips:

  • Bust measurement +2" for comfort
  • XS=32-34", S=34-36" bust
  • High stretch fabrics fit snug

Bottom Size Tips:

  • Hip measurement most crucial
  • Waist-hip difference affects fit
  • 7-9" shorts inseam standard

♂ Men's Fit

Top Size Tips:

  • Shoulder width determines size
  • S=35-37", M=38-40" chest
  • Jerseys run 2" longer

Bottom Size Tips:

  • Inseam varies by height
  • 7-9" shorts for most sports
  • Thigh room for athletes

🧒 Teens/Youth Fit

Growth Considerations:

  • Size up for growing room
  • Adjustable waistbands ideal
  • XS-S for most teens

Sport Specific:

  • +1 size for school teams
  • Stretch fabrics accommodate growth
  • Monitor size every 6 months
